# Photoshop Shortcuts

This markdown file contains a collection of useful Photoshop shortcuts. We will continue to add more as we progress.

## Shortcuts

| Shortcut        | Action           |
|-----------------|------------------|
| **Ctrl + T**    | Scale            |
| **Alt + '+'**   | Zoom In / Zoom Out|
| **Ctrl + R**    | Toggle Rulers    |
